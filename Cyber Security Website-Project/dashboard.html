Welcome to the Security Knowledge Hub

The CIA Triad

Confidentiality: Protects sensitive data from unauthorized access.

Integrity: Ensures data is accurate and unaltered.

Availability: Ensures data is accessible to authorized users.

Authentication

Definition: The process of verifying or proving the user’s identity.

  • Something You Know (Passwords, Passphrases)
  • Something You Have (Tokens, Smart Cards)
  • Something You Are (Biometrics)

Risk Management

Identifies, assesses, and mitigates potential risks to information security.

Threat Actors

  • Insiders (Human error, incompetence, or malicious intent)
  • External Individuals (Cybercriminals, Hackers)
  • Competitors
  • Nation-State Actors
  • AI and Automated Bots

Cryptographic Solutions

  • Symmetric Encryption: Uses the same key for encryption and decryption.
  • Asymmetric Encryption: Uses a public-private key pair for secure communication.
  • Hashing: Converts data into a fixed-length hash for integrity verification.

Public Key Infrastructure (PKI)

PKI manages encryption keys and digital certificates for secure communication.

Encryption for Data Protection

  • Data at Rest – Stored data (e.g., databases, hard drives).
  • Data in Transit – Data traveling over networks.
  • Data in Use – Data currently being processed in memory.
